# Submit Guard — Confirm Before Submit

A lightweight Chrome extension that helps prevent accidental form submissions by adding an optional confirmation step before submit.

- Minimal UI
- Local-only (no backend, no accounts)
- Per-site toggle
- Optional “risky phrase” prompts (e.g. attach / link below / as discussed)

## Status
MVP in progress.

## Development
Load unpacked in Chrome:
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `extension/` folder
